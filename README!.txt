Welcome to the Cube Tutorial!
Prerequesites: Debug Camera

All you need to do is make 8 vertices
in each corner of a cube, exactly the 
same way you would make 3 vertices 
of a triangle, except we will finally
be adjusting the Z value of each vertex,
without leaving it at zero
